# Sleep Early

总想早点睡，却总是不知不觉熬到很晚？Sleep Early 是一款轻量的睡眠打卡工具，帮助你记录入睡时间、查看作息趋势，用更直观的方式管理睡眠习惯。

## 项目简介

Sleep Early 是一个纯前端的睡眠记录小工具，适合用来做日常睡眠打卡和作息观察。

你可以通过它：

- 记录每天的入睡时间
- 区分早睡、一般、熬夜状态
- 查看最近的入睡趋势
- 按本周、本月、年度查看统计结果
- 使用浅色 / 深色主题切换不同界面风格

## 功能特性

- 每日睡眠打卡
- 历史记录补录、修改、删除
- 早睡 / 一般 / 熬夜分类统计
- 睡眠趋势折线图
- 本周 / 本月 / 年度统计切换
- 最早 / 最晚入睡时间统计
- JSON / CSV 导出
- JSON 数据导入
- 本地存储，无需后端服务
- 支持深色模式

## 页面说明

### 1. 时间设定

可自定义两个时间阈值：

- 早睡时间
- 熬夜时间

系统会根据设定时间自动判断每天的记录属于哪一种状态。

### 2. 日历记录

- 用日历查看每天的睡眠记录
- 不同颜色代表不同作息状态
- 支持点击日期补录或编辑

### 3. 趋势图

- 支持查看 7 天、14 天、30 天和全部数据
- 用折线展示入睡时间变化
- 鼠标悬停或手势滑动时可查看具体日期和时间

### 4. 统计视图

- 查看早睡、一般、熬夜的占比与天数
- 查看平均入睡时间
- 查看总打卡天数
- 查看最早 / 最晚入睡时间

## 数据存储

项目当前使用浏览器本地存储保存数据，不依赖后端服务。

默认保存内容包括：

- 时间设置
- 睡眠打卡记录
- 主题模式

## 使用方式

这是一个纯前端页面项目，直接打开 `sleep.html` 即可使用。

### 本地预览

1. 打开项目目录
2. 用浏览器打开 `sleep.html`
